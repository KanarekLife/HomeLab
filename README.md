# HomeLab

Latest revision of my personal HomeLab setup.

## Plan

Plan is to use [Talos Linux](https://talos.dev/) as the main OS for the server. It will host all workloads, utilise GTX 1080 for media transcoding and store data on a ZFS pool (2 x 4TB in a mirror). Eventually this setup will be expanded with a cloud-hosted nodes for redundancy and failover.
